# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial PyPI release
- Python implementation of ctfscan CLI tool
- Flag detection with regex patterns (flag{}, FLAG{}, CTF{})
- Base64 decoding functionality
- Hex string decoding capability
- Advanced scanning with exiftool, binwalk, and strings
- Recursive directory scanning
- Cross-platform compatibility (Python 3.6+)
- Comprehensive documentation and examples

### Changed
- Migrated from bash script to Python for better cross-platform support

### Fixed
- Improved error handling and reporting
